<h1>自己紹介クイズゲーム</h1>
<h2>コンセプト</h2>
<p>製作者がどんな人物なのかゲームを用いて、知ってもらえるようなアプリ</p>
<h2>使用言語</h2>
<ul>
    <li>HTML5</li>
    <li>CSS3(Sass)</li>
    <li>Javascript</li>
    <li>axios</li>
</ul>
<h2>アピールポイント</h2>
<p>下層ページは制作せずに一つのファイルのみで完結するようにしました。また、Annict APIを使用して、アニメ情報を取得。クイズを解き終えたら、私が現在見ているアニメ情報を表示、画像、Twitterのロゴをクリックするとそれぞれのサイト、アカウントに遷移するようになっている</p>
<h2>制作期間</h2>
<p>4日間</p>
